# MROR

MROR ist ein System zur Darstellung von Zeit‑Raster (3×11),
GHOST‑SCAN‑Lagen und RESPO‑Cluster‑Analyse.

## Dateien

- index.html – GHOST‑SCAN
- time_3x11_scroll_zoom.html – Zeit‑Raster mit Zoom + Move
- fin.e.js – Kernel (GHOST, RAUM, MOVE, TOOLOMAT)
- rror.js – MROR‑Helper
- RESPO/ – Cluster‑Daten

## Funktion

MROR lädt RESPO‑Cluster und zeigt:

- ID
- LAGE
- STATUS
- RESULT
- OUT
- WAHRHEIT
- EDIT

MROR ist kompatibel mit RAUM‑ID.
